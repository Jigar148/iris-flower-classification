🌸 Iris Flower Classification

This machine learning project uses a Random Forest Classifier to categorize Iris flowers into three species.

📂 Project Overview

  • 🔍 Dataset: Built-in Iris dataset from sklearn
  • 🧠 Model: RandomForestClassifier
  • 📝 Output: Accuracy and classification report

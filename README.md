# 🌸 Iris Flower Classification using Random Forest

This project classifies Iris flowers into three species (`setosa`, `versicolor`, `virginica`) using a Random Forest classifier.

## 🔍 Dataset
We use the built-in Iris dataset from `sklearn.datasets`.

## 📊 Features
- sepal length (cm)
- sepal width (cm)
- petal length (cm)
- petal width (cm)

## 🧠 Model
- **RandomForestClassifier** from scikit-learn.
- Accuracy evaluated on 20% test split.

## 📝 Output
Classification Report is printed, along with model accuracy.
